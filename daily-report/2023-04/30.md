## 学んだこと（新しい気付き、学び）
- HEADとは現在のブランチを指すポインタ
- detached HEAD は､ブランチを介さずコミットを参照してる状態｡
- git reset HEAD と git restore --staged はどちらもインデックスを取り消す｡
